RepeatZone Node — iterative geometry processing

The RepeatZone executes a body function N times, threading the output of each iteration as the input to the next. Unlocks algorithms impossible in a static DAG — fractal geometry, mesh relaxation, iterative growth.

Left: Fractal displacement (subdivide + noise each pass)
Centre: Mesh relaxation (Laplacian smoothing passes)
